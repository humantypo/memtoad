---
name: context-capture
description: Capture current work-in-progress state to diary/session_context.md for continuity across compact events. Use after significant progress, before context might be lost, or to check what was previously being worked on.
---

Use the Agent tool to spawn the `context-capture` subagent with this prompt:

"Capture the current session state. Review recent conversation context, update diary/session_context.md with what is actively in progress, what has just been completed, and what is next. Mark any previously active items that are now done."
